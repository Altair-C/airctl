# airctl

No-panel Hysteria2 deployment framework.

## Features

- Hysteria2 official server
- CLI menu management
- User add/delete/list
- Shadowrocket/sing-box link generation
- QR code generation
- systemd service management

## Install

    sudo ./install.sh

## Manage

    sudo airctl

## Ports

- 22/tcp SSH
- 8443/udp Hysteria2
